# 论文叙事重构模板（Mode B）

> 适用于：已有实验结果，但论文写成了"方法介绍"，需要重构为"回答问题的故事"。

---

## 第一步：诊断现有叙事

填写下面的框，看你的论文现在是什么状态：

### 现有叙事（方法导向）

> 我们提出了 ___（方法名）___，通过 ___（技术手段）___，在 ___（任务）___ 上取得了 ___（指标）___。

**问题所在**：这是在描述"做了什么"，不是在回答"为什么会这样"。

---

## 第二步：挖掘被埋藏的谜题

从你的实验结果里找答案（谜题通常藏在这里）：

| 挖掘角度 | 你的发现 |
|---------|---------|
| 消融实验：哪个因素贡献最大/最意外？ | |
| 对比实验：最大的性能差距是什么？ | |
| 失败案例：模型在哪种情况下表现最差？ | |
| 意外发现：有没有违反直觉的结果？ | |

**判断标准**：说出来，别人第一反应是"为什么？"还是"哦"？
→ 第一个就是你的谜题。

---

## 第三步：将谜题转化为研究问题

用"有的……有的……"句型重新提问：

> 同样是 ___（控制变量）___，为什么有的 ___（情况A）___，有的却 ___（情况B）___？

或者更直接：

> ___（现象）___ 背后的机制/原因是什么？

**检查**：这个问题里有没有"？"，别人会不会想知道答案。

---

## 第四步：重构叙事

### 新叙事（问题导向）

填写下面三段框架：

**第一段：谜题（让读者想问"为什么"）**
> ___（现有方法/常识）___ 表明 ___（预期）___，然而 ___（你的观察或发现）___，这与预期相悖。这引发了一个核心问题：___？

**第二段：研究问题（明确的疑问句）**
> 本文聚焦于三个子问题：
> ① 为什么有的 ___，有的却 ___？
> ② 什么因素影响了 ___？
> ③ 这些因素通过什么机制起作用？

**第三段：贡献（直接回答研究问题）**
> 通过 ___（方法/实验）___，我们发现：
> ① ___（回答子问题①）___
> ② ___（回答子问题②）___
> ③ ___（回答子问题③）___

---

## 第五步：叙事自检

- [ ] 引言第一段有没有让人想问"为什么"的谜题？
- [ ] 研究问题是疑问句，带"？"？
- [ ] 研究问题是否足够聚焦（不是大议题）？
- [ ] 贡献是否直接回答了研究问题（而不是描述方法）？
- [ ] "从小到大"：你的具体发现，有没有关联到更大的问题或意义？

---

## 输出卡片格式

```
【原叙事】
我们提出了___，在___上取得了___。

【找到的谜题】
___（现象），与___（预期/常识）不符。

【重构的研究问题】
为什么___？（有问号的疑问句）

【引言建议结构】
段1: 谜题 → 引发好奇
段2: 研究问题 → 明确聚焦
段3: 本文贡献 → 直接回答

【Discussion建议】
从具体发现扩展到：___（更大问题的意义）
```
