# 沙漏收口工作表

## 使用说明
用四个维度逐步缩小研究范围，直到你能驾驭为止。

## 第一步：缩小研究对象（Who）

| 层级 | 范围 | 你的选择 |
|------|------|----------|
| 大 | 所有XX | |
| 中 | 某类XX | |
| 小 | 特定XX | |

示例：
- 大学生 → 本科生 → 某专业本科生 → 大一新生
- 深度学习模型 → 大语言模型 → 开源大语言模型 → LLaMA系列

---

## 第二步：缩小时间范围（When）

| 选项 | 时间范围 |
|------|----------|
| 全时段 | |
| 近N年 | |
| 特定时期 | |

示例：
- 2020-2025
- 疫情期间（2020-2022）
- 某版本发布后

---

## 第三步：缩小地域/场景（Where）

| 选项 | 范围 |
|------|------|
| 全球/全国 | |
| 特定地区 | |
| 特定场景 | |

示例：
- 北京某高校
- 医疗影像场景
- 中文环境

---

## 第四步：缩小研究主题（What）
