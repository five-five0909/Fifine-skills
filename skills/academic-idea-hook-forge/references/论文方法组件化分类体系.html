论文方法组件化笔记体系

一个可以长期使用的读论文分类坐标系：把论文里的好方法拆成「组件、公式、任务、实验」四类信息，方便以后写综述、找创新点、改模型。

📌 适用：深度学习 / 遥感 / 高光谱 / 土壤 / Mamba / SSM / PINN 🧩 核心：A-J 拆模型，K-T 拆论文，U 标记公式创新 🗂️ 用法：看论文时直接归类摘抄

0. 这个体系怎么用

💡
一句话：以后你读论文，不是只记"这篇论文叫什么"，而是问：它改了模型哪一步？用了什么公式？解决什么痛点？能不能迁移到你的土壤/遥感/高光谱任务里？
层级你要判断的问题结果放到哪里例子
第一层：论文类型它是方法、数据集、综述、理论、应用，还是系统部署？K-T 分类Survey 放 S；数据集放 K；部署论文放 Q
第二层：模型组件它改了输入、特征、交互、记忆、融合、训练、加速，还是知识约束？A-J 分类Mamba 放 D/E/I；S2Mamba 放 A/D/G
第三层：公式标签它是不是主要改了一个公式、算子、矩阵分解、损失函数？U 标签PINN 放 U6；Hyena 放 U4/U5
第四层：为我所用它能不能用到我的方向？需要什么实验来证明？个人应用笔记CNN + Mamba + SoilML Loss

1. A-T 总分类

最终建议用 A-T 二十类体系：A-J 负责拆模型组件，K-T 负责拆论文类型和研究价值。额外加一个横向标签 U：公式/算子/损失函数创新。

编号大类主要记录什么看论文时摘什么
A输入编码类原始数据如何变成 token / 特征Patch、Token、位置编码、扫描方式、光谱编码
B局部特征类局部纹理、邻域、短程模式CNN、大核卷积、多尺度卷积、局部窗口
C全局交互类不同位置、token、模态之间如何交互Attention、Cross-Attention、Linear Attention
D长程建模类如何处理长序列、长上下文、高分辨率图像SSM、Mamba、Hyena、RetNet、RWKV、xLSTM
E记忆更新类历史状态如何写入、遗忘、读取、编辑Gate、Selective Update、Delta Update
F选择/注意力类哪些通道、空间、时间、波段重要SENet、CBAM、ECA、光谱注意力、状态门控
G特征融合类多路特征如何合并Add、Concat、Gate、Cross-Attention、Mixture Gate
H训练稳定类如何让模型训得稳、收敛快、不过拟合Residual、Norm、Init、Regularization、Aux Loss
I计算加速类如何更快、更省显存、更适合硬件Flash、Scan、FFT、结构化矩阵、KV Cache 压缩
J任务/知识约束类如何让模型符合物理、土壤、光谱、领域规律PINN、SoilML、SCORPAN、光谱先验、不确定性
K数据与基准类数据集、benchmark、标注方式数据来源、样本量、标注规则、任务设置
L评价指标类如何评价模型好坏Accuracy、F1、mIoU、RMSE、R²、ECE、复杂度
M优化算法类优化器、学习率、训练策略、损失权重AdamW、scheduler、loss reweighting、gradient trick
N学习范式类如何学习：监督、自监督、半监督、迁移、多任务Pretraining、Contrastive、SSL、Fine-tuning、MTL
O可解释性类模型为什么这么预测SHAP、Attention Map、Band Importance、Feature Attribution
P可信学习类鲁棒性、不确定性、分布外、校准UQ、OOD、Robustness、Calibration
Q部署系统类模型如何上设备、上硬件、上系统量化、剪枝、蒸馏、边缘部署、NPU/GPU 优化
R理论分析类复杂度、收敛性、表达能力、泛化边界Proof、Theorem、Complexity、Stability Analysis
S综述地图类领域路线图、taxonomy、survey分类体系、发展脉络、开放问题
T应用任务类具体任务场景土壤有机碳、遥感分类、高光谱、气象、医学图像

2. A-J 模型组件分类

这一部分是你以后看"方法类论文"最常用的分类。它对应模型从输入到输出的各个环节。

原始数据 ↓ A 输入编码 ↓ B 局部特征 ↓ C 全局交互 / D 长程建模 ↓ E 记忆更新 / F 选择注意力 ↓ G 特征融合 ↓ H 训练稳定 ↓ I 计算加速 ↓ J 任务/知识约束 ↓ 输出：分类 / 回归 / 分割 / 预测 / 生成

A. 输入编码类

子类通俗理解典型技术适合任务可画图方式
Patch Embedding把图像切成小块ViT Patch、Overlap Patch图像、遥感、医学图像Image → Patch → Token
Tokenization把输入变成 tokenWord Token、Pixel Token、Band Token文本、图像、高光谱Raw Input → Token Sequence
位置编码告诉模型每个 token 在哪里RoPE、ALiBi、2D PE、Relative PENLP、视觉、遥感Token + Position
扫描方式模型按什么路线读数据Cross Scan、双向扫描、蛇形扫描Vision Mamba、高光谱2D/3D Data → Scan Path
光谱编码把波段当成序列Band Tokenization、Spectral Embedding高光谱、土壤反演Bands → Spectral Tokens
图编码把对象当成节点和边Node/Edge Embedding、Graph PE点云、交通、图数据Graph → Node/Edge Features

C-D-E. 全局交互、长程建模、记忆更新

类别子类/模型通俗理解核心优势主要记录字段
C 全局交互标准 Attention每个 token 都能看其他 token内容寻址强、表达力强复杂度、Q/K/V、注意力范围
C 全局交互Cross-Attention一种特征去查询另一种特征适合多模态、多分支融合Query 来自哪一路，Key/Value 来自哪一路
C 全局交互Linear Attention不显式构造完整 L×L 注意力矩阵降低复杂度核函数、递推形式、近似误差
D 长程建模S4 / S5 / SSM把历史压缩进状态长序列、理论基础好状态矩阵 A、离散化、scan/conv 实现
D 长程建模Mamba让状态根据输入选择性更新线性复杂度、状态小选择机制、selective scan、状态大小
D 长程建模Mamba-2 / SSD把 SSM 与 attention 用结构化矩阵统一更快、更硬件友好semiseparable matrix、block decomposition
D 长程建模Hyena长卷积 + 门控替代注意力超长上下文高效long convolution、gating、FFT
E 记忆更新门控更新用门控制写入和遗忘减少状态污染、缓解梯度问题gate 公式、写入/遗忘对象
E 记忆更新选择性更新当前输入决定哪些信息保留内容选择能力强A/B/C/Δ 是否输入依赖
E 记忆更新Delta 更新像编辑记忆表一样更新状态更精细地擦写压缩记忆erase/write 机制、关联记忆形式

F-G. 选择注意力与融合类

类别子类代表模块解决痛点可迁移方向
F通道注意力SENet、ECA、SKNet哪些通道更重要遥感图像、医学图像、CNN backbone
F空间注意力CBAM、DANet、SimAM哪些空间位置更重要检测、分割、变化检测
F时间注意力PAM、CT-MSA、Trend-aware哪些时间片更关键时间序列预测、传感器数据
F光谱注意力Spectral Gate、Band Attention哪些波段更重要高光谱、SOC 预测
GAdd / ConcatResNet Add、U-Net Concat基础融合所有任务
GGate FusionGFU、Mixture Gate动态决定不同分支占比空间-光谱融合、局部-全局融合
GCross-Attention FusionCross-Attn一种模态主动查询另一种模态多模态、遥感+文本、图像+光谱
GKnowledge FusionSoilML、Physics-guided ML把领域知识融入模型土壤、气象、水文、科学计算

H-I-J. 训练稳定、计算加速、知识约束

类别子类通俗理解典型技术记录重点
H残差连接给梯度一条捷径Residual、Skip Connection放在 block 内还是 block 外
H归一化稳定特征分布LayerNorm、RMSNorm、BatchNormPreNorm/PostNorm、归一化维度
H初始化让模型一开始别乱HiPPO、stable init状态矩阵 A 如何初始化
I注意力加速避免完整 L×L 矩阵Sparse/Linear Attention、FlashAttention时间复杂度、显存、是否改变数学结果
I状态递推加速把一步步递推并行起来Parallel Scan、Selective Scan、SSDscan 类型、是否硬件友好
I结构化矩阵让矩阵有特殊结构，便于快速计算Diagonal、DPLR、Semiseparable矩阵结构、复杂度下降
J物理约束预测不能违反物理规律PINN、PIMLPDE residual、边界条件、守恒项
J土壤知识约束预测要符合土壤规律SoilML、SCORPAN土壤先验、深度一致性、合理范围
J光谱先验波段有连续性和物理含义Spectral Smoothness、Band Prior波段重要性、吸收峰、噪声波段

3. K-T 论文类型分类

🧭
A-J 解决"模型怎么拆"，K-T 解决"论文怎么读"。有些论文不是提出新模块，而是提出数据集、指标、理论、部署方案、综述地图，这时用 K-T。
编号论文类型判断标准摘抄重点常见价值
K数据与基准主要贡献是新数据集或 benchmark数据规模、标注方式、任务设置、划分方式以后实验可用
L评价指标主要讨论如何评价模型指标公式、适用场景、优缺点写实验部分可用
M优化算法主要改 optimizer、scheduler、loss weighting更新公式、收敛速度、训练稳定性提升训练效果
N学习范式自监督、半监督、迁移、多任务、预训练预训练任务、伪标签、迁移方式小样本任务很重要
O可解释性解释模型为什么这么预测解释方法、可视化、波段/特征重要性科学任务加分
P可信学习鲁棒性、不确定性、OOD、校准不确定性估计、分布外检测、鲁棒测试让模型更可靠
Q部署系统量化、剪枝、蒸馏、硬件加速模型大小、速度、功耗、设备平台边缘智能感知可用
R理论分析主要是证明、复杂度、泛化、稳定性定理结论、假设条件、证明思路增强论文严谨性
S综述地图survey、taxonomy、路线图分类方式、发展脉络、开放问题写开题/综述可用
T应用任务主要落在具体场景任务定义、数据、评价、实际痛点连接你的研究方向

4. U：公式 / 算子 / 损失函数创新标签

很多高级论文的核心不是复杂结构图，而是一个关键公式：它改变了状态更新、矩阵计算、损失约束或信息流动方式。因此 U 不是单独文件夹，而是横向标签：一篇论文可以同时属于 D/E/I，又带 U1/U3 标签。

编号公式创新类型改的是哪里典型例子摘抄重点
U1状态更新公式h_t 如何由 h_{t-1} 和 x_t 得到SSM、Mamba、xLSTM、DeltaNet写入、遗忘、读取规则
U2注意力公式QKᵀV 怎么改Linear Attention、RetNet、HiLo复杂度、表达能力、近似方式
U3矩阵参数化公式A、K、Toeplitz、低秩矩阵怎么表示S4、Mamba-2、Hyena为什么更快、更稳、更省显存
U4卷积/滤波公式长卷积、频域滤波怎么做Hyena、TCN、ModernTCN感受野、FFT、长程依赖
U5门控公式哪些信息通过，哪些被压制GLU、Mamba Gate、SENet门控对象：通道/空间/状态/波段
U6损失函数公式Loss 如何组成PINN、SoilML、Contrastive Loss每一项 loss 的物理或任务意义
U7归一化/优化公式梯度和特征怎么稳定LayerNorm、RMSNorm、AdamW训练稳定性、收敛速度
U8采样/权重公式样本、token、collocation point 如何加权RAR、Focal Loss、Hard Mining难样本、边界点、稀缺样本如何处理
⚠️
判断公式创新是否有价值：它是否解决真实痛点？是否更快、更稳、更准、更符合物理/土壤规律？是否有复杂度分析和消融实验？如果只是把符号写复杂，价值不大。

5. 读论文归类流程

Step 1：判断论文类型 方法 / 数据集 / 综述 / 理论 / 应用 / 系统 / 优化 / 可信学习 Step 2：判断模型改了哪一步 输入编码 → 局部特征 → 全局交互 → 长程建模 → 记忆更新 → 选择注意力 → 融合 → 训练稳定 → 计算加速 → 知识约束 Step 3：判断是否有公式创新 状态更新 / 注意力 / 矩阵参数化 / 长卷积 / 门控 / 损失函数 / 优化 / 采样 Step 4：判断能否为我所用 能不能迁移到：土壤有机碳预测 / 高光谱分类 / 遥感分割 / 时序传感器 / 边缘智能硬件 Step 5：记录证据 消融实验 / 复杂度分析 / 可解释性 / 不确定性 / 多数据集验证

6. 速查表

6.1 创新类型速查

你看到的创新它通常属于强创新的判断标准弱创新的风险
多加一个注意力模块F / C明确解决噪声、尺度、光谱、空间选择问题只是堆模块
改一个状态更新公式E / U1更会记、更会忘、更稳定、更适合长序列只是换符号
改一个矩阵结构I / U3复杂度或显存显著下降，数值稳定性提升数学包装，实验无提升
改一个 lossJ / U6每一项都有物理/任务含义，消融有效损失项乱加，权重难调
设计多分支融合G每个分支有明确分工，融合自适应concat 后参数变多
设计轻量模型I / Q速度、显存、参数、精度都报告清楚只报参数不报真实速度

6.2 实验证明速查

你提出的东西必须做什么实验最好再补什么
新模块消融实验：去掉模块是否下降可视化模块作用
新公式与旧公式对比 + 复杂度分析理论解释或稳定性分析
新融合方式与 Add / Concat / Attention 融合对比展示不同分支权重变化
新加速算法速度、显存、FLOPs、吞吐量不同序列长度/分辨率下曲线
新知识约束有/无约束对比合理性、误差分布、可解释性
新遥感/土壤模型多数据集、多指标、消融波段重要性、空间可视化、不确定性
