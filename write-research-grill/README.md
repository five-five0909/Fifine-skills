# Write Research Grill Skill

A skill for rigorous pre-writing clarity: before you start writing, force your topic, audience, evidence, structure, and argument to become explicit and testable.

## What it does

This skill turns vague writing intentions into structured pressure tests:

- topic triage and selection
- audience and reader analysis
- material and source interrogation
- structure and outline pre-review
- argument stress testing
- full pre-writing state mapping
